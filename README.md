# 🚀 Agentic RAG From Scratch

A complete hands-on learning repository for building **Agentic Retrieval-Augmented Generation (RAG)** systems using **LangGraph**, **Groq**, **ChromaDB**, **Sentence Transformers**, and **Tavily Search**.

This repository covers everything from the fundamentals of RAG to production-ready Agentic RAG workflows.
